<div class="mb-0">
  <h2 class="text-[18px] font-medium text-text-primary mb-2">Memory Class</h2>
  <p class="text-[14px] text-text-secondary leading-[1.7] mb-4">The `Memory` class allows you to manually store and retrieve long-term context for your users.</p>
  
  <h3 class="text-[15px] font-medium text-text-primary mt-6 mb-2">mem.set()</h3>
  <CodeBlock lang="python" code={`mem.set(user_id: str, key: str, value: str)`} />
  <p class="text-[14px] text-text-secondary leading-[1.7] mt-3 mb-6">Stores a key-value pair for a specific user. If the key already exists, it will be overwritten.</p>

  <h3 class="text-[15px] font-medium text-text-primary mb-2">mem.get()</h3>
  <CodeBlock lang="python" code={`mem.get(user_id: str, key: str = None) -> Union[str, dict]`} />
  <p class="text-[14px] text-text-secondary leading-[1.7] mt-3 mb-6">Retrieves memory for a user. If `key` is provided, returns a string. If `key` is omitted, returns a dictionary of all keys for that user.</p>

  <h3 class="text-[15px] font-medium text-text-primary mb-2">mem.delete()</h3>
  <CodeBlock lang="python" code={`mem.delete(user_id: str, key: str) -> bool`} />
  <p class="text-[14px] text-text-secondary leading-[1.7] mt-3">Deletes a specific memory key for a user. Returns True if successful.</p>
</div>
