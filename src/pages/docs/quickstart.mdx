  <div class="text-[11px] uppercase text-text-muted mb-1">03</div>
  <h2 class="text-[18px] font-medium text-text-primary mb-2">Track your AI calls</h2>
  <p class="text-[14px] text-text-secondary leading-[1.7] mb-4">Add the @track decorator to any function that calls an LLM. Mindphor captures the input, output, and latency automatically.</p>
  
  <CodeBlock lang="python" code={`from mindphor import track
import openai

@track(
    api_key="mp_your_key_here",
    model="gpt-4",
    user_id="user_123"
)
def ask_ai(question: str) -> str:
    response = openai.chat.completions.create(
        model="gpt-4",
        messages=[
            {"role": "user", "content": question}
        ]
    )
    return response.choices[0].message.content

# This call is now tracked and scored
answer = ask_ai("What is the refund policy?")`} />
</div>

  <div class="text-[11px] uppercase text-text-muted mb-1">04</div>
  <h2 class="text-[18px] font-medium text-text-primary mb-2">Track user memory</h2>
  <p class="text-[14px] text-text-secondary leading-[1.7] mb-4">Use the Memory class to store and monitor user context. Mindphor automatically detects when memory keys go stale or contradict each other.</p>
  
  <CodeBlock lang="python" code={`from mindphor import Memory

mem = Memory(api_key="mp_your_key_here")

# Store user context before AI call
mem.set(
    user_id="user_123",
    key="plan",
    value="pro"
)

# Retrieve all memory for a user
keys = mem.get(user_id="user_123")

# Delete a specific key
mem.delete(user_id="user_123", key="plan")`} />
</div>
