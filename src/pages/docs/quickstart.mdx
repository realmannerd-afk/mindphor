</p>

<CodeBlock lang="bash" code={`pip install mindphor`} />

<h2 class="text-[18px] font-medium text-text-primary mb-4 mt-12">2. Configuration</h2>
<p class="text-[15px] text-text-secondary mb-4 leading-relaxed">
  Set your API key as an environment variable. You can find this in your Mindphor dashboard under <strong>Project Settings</strong>.
</p>

<CodeBlock lang="bash" code={`export MINDPHOR_API_KEY="your_api_key_here"`} />

<h2 class="text-[18px] font-medium text-text-primary mb-4 mt-12">3. Usage</h2>
<p class="text-[15px] text-text-secondary mb-4 leading-relaxed">
  Simply import the <span class="bg-bg-elevated px-1.5 py-0.5 rounded-md border border-border-faint text-text-primary">@track</span> decorator and wrap any function that calls an LLM. Mindphor will intercept the call, send the trace to our Edge API, and asynchronously score the output.
</p>

<CodeBlock lang="python" code={`import openai
from mindphor.tracker import track

@track(project_id="proj_123")
def generate_summary(text: str) -> str:
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[{"role": "user", "content": f"Summarize: {text}"}]
    )
    return response.choices[0].message.content

# This call is now tracked and scored on your dashboard!
summary = generate_summary("Mindphor is an AI monitoring platform...")`} />
